# CV &Image Processing App

An interactive **Streamlit-based Computer Vision project** that demonstrates fundamental image processing, filtering, feature detection, and segmentation techniques using OpenCV.

---

##  Live Features

This application allows users to upload an image and apply multiple Computer Vision algorithms:

###  Image Processing

* Resize Image
* Rotate Image
* Invert Colors
* Grayscale Conversion
* Image Scaling

###  Visualization

* Histogram of Pixel Intensities

###  Filtering

* Mean Filter
* Gaussian Filter

###  Feature Detection

* Canny Edge Detection
* Harris Corner Detection

###  Segmentation

* Otsu Thresholding
* K-Means Clustering Segmentation
* GrabCut Foreground Extraction

---

## Technologies Used

* Python 
* Streamlit
* OpenCV (cv2)
* NumPy
* Matplotlib
* Pillow (PIL)
